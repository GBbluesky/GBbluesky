css를 공부해보자(9)-CSS Grid-template area 활용하기

template area를 활용하여 보다 직관적으로 레이아웃을 구성하는 방법을 알게되어 정리한다.

grid-template-areas:

css grid를 다음과 같이 작성한다.


    .container {
      display: grid;
      grid-gap: 3px;
      grid-template-columns: repeat(8, 1fr);
      grid-template-rows: 100px auto 100px;
    }

이렇게 구성할 경우 3 * 8 의 grid area가 구성될 것이다. 이제 여기에 다음과 같은 코드를 추가한다. 


      grid-template-areas:
        "h h h h h h h h"
        "m c c c c c c c"
        "f f f f f f f f"

그리고 자식 태그들의 그리드 속성을 다음과 같이 작성한다.


        .header {
        grid-area: h;
      }
      .menu {
        grid-area: m;
      }
      .content{
        grid-area: c;
      }
      .footer {
        grid-area: f;
      }

그럼 다음과 같이 레이아웃이 구성된다

area의 숫자가 많아지면 활용하기 힘들겠지만, 숫자가 많지 않고 단순한 레이아웃을 짤때는 직관적으로 짤 수 있는 좋은 방법인 것 같다.
