css를 공부해보자(6) - grid layout의 명칭

css grid layout의 기본적인 명칭들을 정리해보았다. Mozelladeveloper라는 곳에 깔끔하게 정리되어 있어 참고하였다.

1. grid lines

grid line은 그리드 칸을 나누는 선이다. "grid-column-start : 1"등 그리드의 칸을 지정할 때 사용한다.

2. grid cell

grid cell은 그리드 한 칸을 의미한다.

3. grdi area

grid area는 여러 grid cell로 이루어진 영역을 말한다.

4. grid track

grid track은 두개의 grid line사이에 있는 영역을 말한다. 가로, 세로 둘다 적용된다. 가로는 "grid row", 세로는 "grid column"이라고 말한다.

5. grid gutter

gird gutter은 grid column, row 사이의 영역을 말한다. grid line과의 차이점이라면 line은 선이지만 grid gutter는 영역, 2차원이다.

6. grid container and grid item

grid container는 그리드 전체를 지칭하는 말. css에서 "display: grid"라는 코드로 표현된다. grid item은 container에 딸린 그리드 영역들이다.
