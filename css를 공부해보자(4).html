웹폰트

그리드를 공부하기에 앞서 잠시 알게된 몇가지를 정리하고 넘어가려 한다. 웹폰트는 폰트를 따로 추가하지 않고 링크를 통해 불러오는 기능이다. 다음과 같이 작동한다. 


      @import url(//fonts.googleapis.com/earlyaccess/notosanskr.css);
      body{
        font-size: 16px;
        font-family: "noto sans KR", sans-serif;
        font-weight: 400;
      }
'noto sans KR'이란 폰트를 다른 방법을 사용하지 않고 그대로 페이지 전체에 적용한다.

티스토리 화면이 깨져서 나타나는 문제

현재까지 모든 포스트를 html&css로 Atom이란 프로그램을 통해 작성한 뒤 올리고 있다. 문제는 html로 작성해서 올리다보니 간혹 내가 작성한 css style이 문서 밖의 태그들에도 속성을 적용시킨다는 것이다. 가장 근본적인 해결책은 style.css문서를 작성해 블로그에 적용시키는 것이겠지만 아직은 그럴 단계가 아니다. 때문에 임시적으로 모든 element에 class를 적용시키는 방법으로 해결하고 있다. 어서 그리드와 그밖의 css기능들을 익혀서 블로그 스킨을 업로드 해야겠다.
