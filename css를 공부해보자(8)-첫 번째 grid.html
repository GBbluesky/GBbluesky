css를 공부해보자(3)-css layout에서 내가 만들고 싶은 레이아웃을 올렸었다. grid를 활용해 레이아웃을 만들었던 과정을 간략히 정리해보도록 하겠다.

2. grid css 작성

앞서 작성한 html를 바탕으로 다음과 같이 grid를 짜보았다. 


    .container{
      height: 100%;
      margin: 10px;
      display: grid;
      grid-template-columns: repeat(11, 1fr);
      grid-template-rows: 50px 300px 400px 50px 150px 200px;
      grid-gap: 3px;
    }
    .header{
      grid-column: 1 / -1;
    }
    .mainimage{
      grid-column: 1 / -1;
    }
    .menu{
      grid-column: 1 / 3;
    }
    .content{
      grid-column: 3 / -2;
    }
    .tag{
      grid-column: 3 / 11;
    }
    .prev1{
      grid-column: 3 / 5;
    }
    .prev2{
      grid-column: 6 / 8;
    }
    .prev3{
      grid-column: 9 / 11;
    }
    .comment{
      grid-column: 3 / 11;
    }

위와 같이 grid를 구성했을때 다음과 같은 화면이 나온다.

그리드를 구성하던 중 실수한 부분이 있다. repeat(11, 1fr);를 repeat (11, 1fr);이라고 작성한 것. 띄어쓰기를 하면 안되는데 띄어쓰기를 하여 적용이 되지 않았다. 코드를 작성할 때 띄어쓰기 하나도 주의해야 겠다.

3. css를 활용하여 꾸미기

그리드는 원하는대로 짜여졌지만 화면이 많이 허전해보인다. 그래서 다음과 같은 css를 추가로 작성하였다. 


    html, body{
      box-sizing: border-box;
      background-color: #F7E967;
      height: 100%
      padding: 20px;
      margin: 0px;
    }
    .container > div{
      display: flex;
      justify-content: center;
      align-items: center;
      font-size: 2em;
      color: #Ffffff;
    }
    .container > div:nth-child(1n){
      background-color: #588F27;
    }
    .container > div:nth-child(2n){
      background-color: #CAFCD8;
    }
    .container > div:nth-child(3n){
      background-color: #A9CF54;
    }
    .container > div:nth-child(4n){
      background-color: #04BFBF;
    }

처음보는 속성들(div:ntn-child(2n))은 "learn CSS grid for free"에서 따왔다. 영상에서 사용한 코드들을 베끼면서 연습하였다. "div:ntn-child(an+b)"는 선택자를 다루는 기법이다. an+b를 수열처럼 활용하여 부모태그 아래있는 자식태그를 선택한다. 2n 일경우 짝수, 3n일 경우 3의 배수번째 자식태그들을 선택한다. 이를 적용하여 최종적으로 만든 화면은 다음과 같다.
