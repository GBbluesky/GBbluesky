css를 공부해보자(2)

css는 선택자와 선언(속성과 값)으로 이루어져있다. css의 선택자들은 다음과 같은 것들이 있다.

*아래 내용은 w3schools의 설명을 정리한 것이다.

Selector(선택자)

Element Selector
직역하자면 요소선택자. element는 html태그를 의미한다. html태그를 직접 선택하여 속성을 부여한다. 예를 들어 
        p {
          text-align: center;
          color: red;
        }
이렇게 입력할 경우 페이지에 있는 모든 p태그는 가운데 정렬, 빨간색으로 표시된다.
Id Selector
id 선택자. 특정 id를 가진 태그만 선택한다. id는 고유한 것이기에 하나의 html문서에는 동일한 id를 가진 태그가 있어서는 안된다. (비유하자면 주민등록번호가 같은 사람이 있으면 안되는 것 처럼) 
        #para {
    text-align: center;
    color: red;
  }
para라는 id를 가진 태그에만 가운데 정렬, 빨간색으로 표시를 적용한다. id는 다음과 같이 부여한다. 
 id="para"
Class Selector
class 선택자는 동일한 class를 가진 태그들을 선택한다. 즉, 다수의 태그들을 선택하기 위해 있는 선택자다. (비유하자면 같은 반 소속인 학생들을 선택하는 것이다) element 선택자보다는 좁고, id 선택자보다는 넓은 범위로 이해된다. 
        .center {
          text-align: center;
          color: red;
        }
css에 이렇게 작성하고 html태그에는 다음처럼 class를 부여한다.
 p class="center"
추가로, 여러 class중 p태그 아래 있는 class만 선택할 수도 있다. 
      p.center {
        text-align: center;
        color: red;
      }
이럴 경우 
h1 class="center"
      p class="center"
위에서 h1에는 적용되지 않고 p에만 적용된다.
Grouping Selectors
여러 선택자들을 그룹으로 묶는다. 다른 종류의 선택자라기 보다는 효율성을 위해 나타내는 방식이라고 이해된다. 예를 들어 p, h1, h2를 가운데 정렬, 빨간색으로 표현하고 싶다면 
        h1{
          text-align: center;
          color: red;
        }
        h2{
          text-align: center;
          color: red;
        }
        p{
          text-align: center;
          color: red;
        }
이렇게 적어야 할 것이다. 하지만 그룹으로 묶는다면 
          h1, h2, p {
            text-align: center;
            color: red;
          }
훨씬 간결하게 속성을 적용할 수 있다.
