CSS를 공부해보자(5)-CSS grid

저번 포스트css를 공부해보자(3)-레이아웃에서 css 그리드를 공부하기로 했었다. 다행히 css 그리드의 기본개념을 쉽게 잘 설명한 페이지를 발견하였다. Learn CSS Grid in 5 Minutes 이 사이트의 내용을 바탕으로 css 그리드의 기초를 정리하고자 한다.

grid는 어떻게 사용하는가

grid는 여러가지 의미로 쓰이지만 css grid에서는 '격자'로 풀이 될 수 있을 것 같다. 쉽게 말하면 페이지를 격자형식으로 레이아웃을 짜주는 거다. 마치 한글에서 표만들기 하듯이 말이다. 그리드를 사용하기 위해서는 다음과 같은 과정이 필요하다.

1. wrapper(부모태그)와 items(자식태그)설정하기

예를 들어 6개의 '구역'을 가진 레이아웃을 짠다고 하자. 그렇다면 그리드를 위해서는 다음과 같이 element를 구성해야 한다.

<div class="wrapper"> <div class="one">1</div> <div class="two">2</div> <div class="three">3</div> <div class="four">4</div> <div class="five">5</div> <div class="six">6</div> </div>

즉 wrapper로 설정된 div태그 아래 6명의 자식이 있는 것이다.

2. grid css 속성 입력

grid는 css에서 다음과 같이 구현된다

.wrapper{ display: grid; }

그러면 다음과 같은 화면이 뜰 것이다.

아직 구체적으로 그리드를 설정하지 않았기에 6개의 div태그가 그대로 쌓여있는 모습이다. 이제 그리드를 설정해보자.


        .wrapper{
          display: grid;
          grid-template-columns: 100px 100px 100px;
          grid-template-rows: 50px 50px;
        }

그럼...짜잔! 다음과 같은 모습으로 페이지가 뜬다.

위에서 설정한 속성의 내용은 "열을 100px 100px 100px (3개)로, 행을 50px 50px (2개)로 설정한다"이다. 즉 칸의 높이가 50px, 너비가 100px 인 3*2 표를 만든것이라고 생각하면 쉽다. 만약 칸을 4*3으로 만들고 싶다면 자식태그를 6개 추가하고

        grid-template-columns: 100px 100px 100px 100px,
        grid-template-rows:50px 50px 50px;

로 설정하면 된다.

3. 좀 더 복잡한 레이아웃 구현

만드려는 레이아웃이 이렇게 단순할 때도 있지만, 대부분은 훨씬 복잡할 것이다. 아직은 입문인 단계여서 다음에 나올 수준밖에 익히지 못했다. 다음의 코드를 css에 추가해보자.


            .one {
              grid-column-start: 1;
              grid-column-end: 4;
            }

그러면 다음과 같이 화면이 뜬다.

위에서 설정한 속성의 의미는 "1번 구역을(class=one이라고 설정했다.) 1 grid line부터 4 grid line까지 차지하도록 하시오"라는 의미이다. grid line은 다음 예제를 보면 이해하기 쉽다.

이를 응용하면 다음과 같이 그리드를 구현할 수 있다.

코드는 다음과 같다


            div.wrapper{
              display: grid;
             grid-template-columns: 100px 100px 100px;
              grid-template-rows: 100px 100px 100px;
            }
            .one{
              grid-column: 1 / 4;
            }
            .three{
              grid-row: 2 / 4;
            }
            .four{
              grid-column: 2 / 4;
            }

공부하기전에는 어려우면 어떡할까 걱정했지만 설명이 워낙 쉽게 잘 되어있어서 그런데로 따라할 수 있었다. 글을 쓴 저자가 다음과 같은 무료 css grid 강좌를 운영하고 있는데 시간이 되는대로 배울 예정이다.
