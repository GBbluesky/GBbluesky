css를 공부해보자(7)-css grid 실습1

그동안 css의 기초적인 부분들을 알아보았다. "Learn CSS grid for free"라는 곳에서 몇가지 배운 몇가지 방법들을 정리한 뒤 앞서 이야기 했던 블로그 layout를 만들어 보려고 한다.

grid 영역 나누기

grid를 '몇 행 몇 열'로 구성할지 정하는 방법은 다음과 같은 것들이 있다.

1. columns (열), rows (행) 명시 

          .container{
            display: grid;
            grid-template-columns: 100px auto 100px;
            grid-template-rows: 50px 50px;
          }
2. repeat 활용하기 

          .container{
            display: grid;
            grid-template-columns: repaet (3, 1fr);
            grid-template-rows: repaet (2, 50px);
          }
3. 행/열로 표현하기

          .container{
            display: gird;
            grid-template: repeat (2, 50px) / (3, 1fr);
          }

가장 간단하게 표현할 수 있는 방법으로 행/열을 헷갈리는 것에 주의하자.

grid item 영역 지정해주기

grid item이 어느정도의 영역을 차지할지를 정하는 방법이다.

1. grid-column(OR row)-start(AND end) 

            .header{
              grid-column-start: 1;
              grid-column-end: 3;
            }
2. start / end 

            .header{
              grid-column: 1 / 3;
            }

둘 다 grid lines를 기준으로 한다. end에 -1을 넣을 경우 맨 마지막 grid line까지 영역을 차지한다는 의미이다. 나중에 큰 페이지를 구축할 때 grid line이 몇개 인지 일일히 새지 않아도 된다. (상단 bar나 좌측 메뉴등을 지정할 때 좋을 것 같다.)
